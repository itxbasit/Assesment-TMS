## Overview
This is the frontend for the Task Management System built using **React 19**, **TypeScript**, and modern React hooks.  
It provides user authentication, task list management, task operations, and sharing functionality with role-based permissions.

## Tech Stack
- React 19+
- TypeScript
- State Management: (Redux / Context API)
- React Router
- Axios
- CSS / Tailwind / Ant Design (as used)

## Features
- User registration & login (JWT-based)
- Protected routes
- View owned and shared task lists
- Create, edit, delete task lists
- Add, update, delete tasks
- Task status management (pending, in progress, completed)
- Share task lists with permission levels (view / edit)
- Responsive UI
